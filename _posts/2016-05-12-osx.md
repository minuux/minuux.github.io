## 使用 OS X 对ubuntu进行管理

[ssh介绍](http://www.ruanyifeng.com/blog/2011/12/ssh_remote_login.html)

```
brew install ssh-copy-id

/*生成公钥与私钥*/
ssh-keygen

ssh-copy-id user@address

ssh user@address
```
